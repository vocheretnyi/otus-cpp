Printing std::tuple as IP.

An address represented as std::list or std::vector containers. The full content of the container is displayed element by element and separated by . (dot character). Items are displayed as is.

The address represented as a std::tuple provided that all types are the same. The complete content is displayed element by element and separated by . (single dot character). Items are displayed as is. If the tuple types are not the same, an error should be thrown when compiling the code.

The address represented as an arbitrary integer type. Output unsigned byte by byte, high byte first, with . (dot character) as delimiter. All bytes of the number are output.
